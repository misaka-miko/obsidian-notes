# Basic Counting
|                     | order matters   | order not matter     |
| ------------------- | --------------- | -------------------- |
| with replacement    | $n^{k}$         | $\binom{n+k-1}{k-1}$ |
| without replacement | $\frac{n!}{k!}$ | $\binom{n}{k}$       |
